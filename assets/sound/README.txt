==================================================
 MaruBatsu サウンド素材フォルダ
==================================================

ここに下記のファイル名で音源を置くと、次回起動時から自動的に再生されます。
（コード変更は一切不要。ファイルが無い ID は無音のままで、エラーにもなりません）

対応形式: .mp3 (推奨) または .wav
　※ .wav を使う場合は sound/sound_manager.cpp の SOUND_FILES[] の
　　 該当パスの拡張子 ".mp3" を ".wav" に変えてください。

--------------------------------------------------
 BGM (ループ再生)
--------------------------------------------------
  bgm_title.mp3    … タイトル / 戦績画面
  bgm_game.mp3     … 対戦 / 報酬画面
  bgm_result.mp3   … リザルト画面

--------------------------------------------------
 SE (単発再生)
--------------------------------------------------
  se_select.mp3    … カーソル移動 / カードのホバー
  se_decide.mp3    … 決定
  se_cancel.mp3    … 戻る / キャンセル
  se_pause.mp3     … ポーズ (将来用)
  se_place.mp3     … 駒の設置
  se_win.mp3       … 勝利
  se_lose.mp3      … 敗北 / 時間切れ
  se_ability.mp3   … アビリティ / ボスギミックの発動
  se_timer_low.mp3 … 思考時間 残りわずか

--------------------------------------------------
 補足
--------------------------------------------------
・参照は実行時カレントディレクトリ基準の "assets/sound/..." 相対パスです。
　通常のデバッグ実行 (プロジェクトフォルダが作業ディレクトリ) ではここを
　そのまま読みます。配布時 (exe単体実行) はビルド後イベントで本フォルダの
　音源が出力先へコピーされます。
・ID とファイルの対応は sound/sound_manager.cpp の SOUND_FILES[] にあります。
　ID を増やす場合は sound/sound_manager.h の enum と両方に追記してください。
