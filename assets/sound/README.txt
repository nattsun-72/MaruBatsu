==================================================
 MaruBatsu サウンド素材フォルダ
==================================================

BGMは bgm/、SEは se/ サブフォルダに下記のファイル名で置くと、
次回起動時から自動的に再生されます。
（コード変更は一切不要。ファイルが無い ID は無音のままで、エラーにもなりません）

対応形式: .mp3 (推奨) または .wav
　※ .wav を使う場合は sound/sound_manager.cpp の SOUND_FILES[] の
　　 該当パスの拡張子 ".mp3" を ".wav" に変えてください。

--------------------------------------------------
 BGM (ループ再生)  →  assets/sound/bgm/
--------------------------------------------------
  bgm/title.mp3    … タイトル / 戦績画面
  bgm/game.mp3     … 対戦 / 報酬 / ボス撃破画面
  bgm/result.mp3   … リザルト画面

--------------------------------------------------
 SE (単発再生)  →  assets/sound/se/
--------------------------------------------------
  se/select.mp3    … カーソル移動 / カードのホバー
  se/decide.mp3    … 決定
  se/cancel.mp3    … 戻る / キャンセル
  se/pause.mp3     … 設定画面を開く (O キー)
  se/place.mp3     … 駒の設置
  se/win.mp3       … 勝利
  se/lose.mp3      … 敗北 / 時間切れ
  se/ability.mp3   … アビリティ / ボスギミックの発動
  se/timer_low.mp3 … 思考時間 残りわずか

--------------------------------------------------
 補足
--------------------------------------------------
・参照は実行時カレントディレクトリ基準の "assets/sound/..." 相対パスです。
　通常のデバッグ実行 (プロジェクトフォルダが作業ディレクトリ) ではここを
　そのまま読みます。配布時 (exe単体実行) はビルド後イベントで本フォルダの
　音源が出力先へコピーされます。
・ID とファイルの対応は sound/sound_manager.cpp の SOUND_FILES[] にあります。
　ID を増やす場合は sound/sound_manager.h の enum と両方に追記してください。
