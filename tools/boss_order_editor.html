手動読み込み（ファイルを開けない環境用）

game_config.json の中身を貼り付けて「取り込む」を押すと現在の順を読み取れます。保存は全文をダウンロードします。
使い方とメモ
・対応ブラウザ（Chrome/Edge）では「開く」から直接 assets/game_config.json を読み書きできます。
・それ以外の環境では、右下にファイルを貼り付け→「取り込む」で現在順を読み、「保存」で全文をダウンロードして既存ファイルに上書きしてください。
・順は重複可（同じボスを2回出すなど）。空にすると、ゲーム側はカタログ定義順で動作します。
・反映タイミング：ゲームはタイトル画面に戻るたびに再読込します。
・ボスを新規追加した場合は、C++ 側 boss_roster.cpp のカタログと、このツール先頭の CATALOG の両方に同じ code で追記してください。
・注意：ラン途中のセーブはボス番号(index)で復元するため、順を変えると再開時の対戦ボスが変わります（設計時の編集を想定）。
